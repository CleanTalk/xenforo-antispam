# xenforo-antispam
Xenforo anti-spam extension.
# Version 2.6

## How to install

1) Upload "CleanTalk" folder to <your_xenforo>/library/ directory.
2) Install "addon-CleanTalk.xml".

## Requirements

* CleanTalk account https://cleantalk.org/register?product=anti-spam
